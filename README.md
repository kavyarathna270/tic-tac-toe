✅ Project Overview (for GitHub description)
This is a Tic-Tac-Toe web app built with React (with Redux state management) that lets two players play a classic 3×3 Tic-Tac-Toe game in the browser.

🧩 What it Includes
Interactive 3×3 board (click to place “O” or “X”)
Win detection logic (detects row/column/diagonal winners)
Turn tracking (switches between players “O” and “X” automatically)
Reset button to restart the game at any time
Uses React functional components and hooks (useState)
Includes a small Redux setup (redux/ folder) (likely for extensibility / state tracking)
